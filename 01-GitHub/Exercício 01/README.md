## 🎯 Objetivo

Este repositório foi criado de forma genérica com o objetivo estrito de exercitar e demonstrar a aplicação prática de recursos visuais e estruturais da linguagem **Markdown** no GitHub. A estrutura simula um ambiente real de documentação profissional para projetos de software.

## ⚡ Recursos Avançados (Desafio Final)

Esta seção agrupa os 5 elementos adicionais da linguagem Markdown que não haviam sido requisitados inicialmente nos exercícios base:

1.  **Texto Riscado:** ~~Este parágrafo representa uma funcionalidade obsoleta que foi cortada do escopo.~~
2.  **Texto Destacado:** Use tags HTML estruturadas para deixar palavras em <mark>evidência visual amarela</mark>.
3.  **HTML dentro do Markdown:** <div align="center"><i>Este bloco de texto foi centralizado horizontalmente utilizando propriedades de tags HTML div.</i></div>
4.  **Detalhes Expansíveis (Dropdown):**
    <details>
    <summary>Clique aqui para expandir e ler o conteúdo oculto</summary>
    <br>
    Você descobriu o conteúdo escondido! Essa estrutura (tags details e summary) é excelente para organizar FAQs ou logs extensos sem poluir o visual do arquivo.
    </details>
5.  **Imagem Clicável (GIF Animado):** O GIF animado abaixo funciona como um link hipertexto configurado por âncora. Clique na imagem para retornar ao início da página:
